# Project Constraints & Guidelines

## General Rules:
1. Always maintain code style consistency.
2. Update task status in task.json after each major step.
3. Log progress and failures in Progress.txt.
4. Clear context after task completion (simulated by focusing on next task).

## Tech Stack:
- Frontend: HTML/CSS/JS (Vanilla)
- Map: Tencent Maps SDK
- UI: Hand-drawn/Cartoon style

## Workflow:
1. Read Task.
2. Implement Feature.
3. Verify.
4. Update Logs.
