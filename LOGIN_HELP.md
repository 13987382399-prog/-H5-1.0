# 解决服务器登录问题

## ❌ 你当前遇到的错误

如果你看到类似这样的错误：
> `curl : 所在位置 行:1 字符: 84`
> `标记“&&”不是此版本中的有效语句分隔符。`

**这说明你正在你自己的电脑（Windows PowerShell）上运行命令，而不是在服务器上！**

请务必先连接到服务器，然后再运行部署命令。

---

## ✅ 正确操作步骤

### 第一步：连接服务器（推荐使用网页版）

由于你在本地遇到了 `Permission denied`（权限被拒绝）的错误，最简单的办法是使用阿里云自带的网页终端：

1.  **登录阿里云控制台**：[https://ecs.console.aliyun.com/](https://ecs.console.aliyun.com/)
2.  找到你的服务器实例 `8.137.59.55`。
3.  点击右侧的 **远程连接** 按钮。
4.  选择 **Workbench远程连接**（通常是第一个选项）。
5.  **登录**：
    *   用户名：`root`
    *   密码：如果你不知道或忘记了密码，请先点击控制台上的 **“重置实例密码”**，设置一个新密码（如 `Kunming@2024!`），然后**重启服务器**。

### 第二步：在服务器中运行脚本

当你看到黑色的终端窗口，并且命令提示符是 `root@...` 开头时，说明你已经成功进入服务器了。

**此时，复制并运行以下命令：**

```bash
curl -O https://raw.githubusercontent.com/13987382399-prog/-H5-1.0/master/setup.sh && chmod +x setup.sh && ./setup.sh
```

### 常见问题排查

*   **问题：我重置了密码，但还是连不上？**
    *   **解决：** 重置密码后，必须在控制台点击 **“重启”** 按钮，新密码才会生效。

*   **问题：网页版终端打不开？**
    *   **解决：** 尝试使用 VNC 连接（远程连接选项中的另一个），或者检查你的浏览器是否拦截了弹窗。
